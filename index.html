Setup

All the code examples must go into your init.lua.

local HYPER={'ctrl','alt','cmd','shift'}
local function bind(mod,key,fn,msg,dur)
  hs.hotkey.bind(mod,key,function()
    fn() hs.alert(msg,dur or 2)
  end)
end

Arrange windows on a grid

On pressing hyper-r, move/resize the focused window on a 5x4 grid:

hs.grids.setGrid'5x4'
bind(HYPER,'r',hs.grids.show,'move/resize window')

Switch between window layouts

Still using just Terminal.app for this example, save a window layout with ctrl-cmd-shift-1..4, restore it with ctrl-cmd-1..4:

local terminal_wl=hs.windowlayouts.new'Terminal'
for i=1,4 do
  local c=tostring(i)
  bind({'ctrl','cmd','shift'},c,function()
    terminal_wl:saveLayout('terminal'..c) 
  end,'Saved layout '..c)
  bind({'ctrl','cmd'},c,function()
    terminal_wl:applyLayout('terminal'..c)
  end,'Layout '..c)
end

Tile windows

Too many browser windows? Quickly tile them with ctrl-alt-t for easier management, then navigate with ctrl-alt-w/a/s/d:

local browsers_wt=hs.windowtiling.new{'Safari','Google Chrome'}
bind({'ctrl','alt'},'t',function()browsers_wt:toggleTiling()end,'Toggle tiling')
bind({'ctrl','alt'},'w',function()browsers_wt:focusNorth(false,true)end,'Focus ↑')
bind({'ctrl','alt'},'s',function()browsers_wt:focusSouth(false,true)end,'Focus ↓')
bind({'ctrl','alt'},'a',function()browsers_wt:focusWest(false,true)end,'Focus ←')
bind({'ctrl','alt'},'d',function()browsers_wt:focusEast(false,true)end,'Focus →')

Automatic tiling

Let's have terminal and Sublime Text windows tile automatically as we add/remove windows:

local term_subl_wt=hs.windowtiling.newAutoTile{'Terminal','Sublime Text'}

Keyboard-only expose

Switch focus and close windows without lifting your hands, with hyper-e:

local expose=hs.expose.new(hs.windowfilter.new(nil,true,nil))
bind(HYPER,'e',function()expose:expose()end,'Expose')

Get that AXSystemDialog

Focus those pesky not-quite-modal system dialogs - even after they've been buried under 20 layers of windows - with hyper-m:

local system_ww=hs.windowwatcher.new{'UserNotificationCenter','SecurityAgent'}
bind(HYPER,'m',function()
    local w=system_ww:getWindows()[1] 
    if w then w:focus() end 
  end,'Focus modal dialog')
